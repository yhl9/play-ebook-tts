### 🔧 技术文档

#### 配置系统
- **主配置文件**: `config.json`
- **引擎配置**: `configs/engines/registry.json`
- **参数配置**: `configs/engine_parameters.json`

#### 日志系统
- **日志级别**: 通过 `config.json` 中的 `advanced.log_level` 配置
- **日志文件**: `logs/app.log` 和 `logs/error.log`
- **调试模式**: 设置 `log_level: "DEBUG"`

### 📖 快速开始

#### 1. 选择 TTS 引擎
根据需求选择合适的 TTS 引擎：
- **高质量在线**: Edge TTS
- **离线高质量**: Piper TTS  
- **快速测试**: pyttsx3

#### 2. 配置语音设置
在应用程序中配置：
- 选择 TTS 引擎
- 选择语音类型
- 调整语音参数

#### 3. 开始使用
- 导入文本文件
- 设置处理参数
- 生成音频文件

### 🛠️ 故障排除

#### 常见问题
1. **语音不可用**: 检查语音包安装
2. **模型下载失败**: 检查网络连接
3. **语音质量差**: 调整参数设置
4. **性能问题**: 优化资源配置

#### 调试方法
1. **启用调试日志**: 设置 `log_level: "DEBUG"`
2. **查看日志文件**: 检查 `logs/` 目录
3. **测试语音功能**: 使用测试功能
4. **检查配置**: 验证配置文件
